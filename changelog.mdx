## Recent Updates

Stay current with the latest releases. You review changes here to understand new capabilities and required actions for your documentation site.

<Update label="2024-10-20" description="v1.3.0" tags={["feature", "improvement"]}>

- Added support for custom brand colors. Set your `{brandColor}` in the config to match `#3B82F6`.

## Improvements

- Optimized Mermaid diagram rendering for faster load times.
- Enhanced search indexing to cover nested Expandable content.

</Update>

<Update label="2024-10-10" description="v1.2.1" tags={["bugfix", "security"]}>

- Fixed parsing errors with nested code blocks using 4 backticks.
- Resolved JSX attribute validation for arrays like `tags={["feature"]}`.

## Security

- Patched potential vulnerabilities in external iframe embeds.

</Update>

## Upgrade Instructions

You upgrade Duyên Phan Documentation by following these steps. Each release includes migration guides within the notes above.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Save your current `docs/` directory and config files.

    <Callout kind="tip">
      Use Git to create a branch: `git checkout -b upgrade-v1.3.0`.
    </Callout>
  </Step>
  <Step title="Update Package" icon="package">
    Install the latest version across platforms.

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install duyen-phan-docs@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```yarn
        yarn add duyen-phan-docs@latest
        ```
      </Tab>
      <Tab title="pnpm" icon="package">
        ```bash
        pnpm add duyen-phan-docs@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Rebuild and Deploy" icon="rocket">
    Run the build command and deploy.

    ```bash
    npm run build
    npm run deploy
    ```
  </Step>
</Steps>
